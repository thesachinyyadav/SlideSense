# Changelog

All notable changes to SlideSense will be documented in this file.

## [1.0] - 2026-02-22

### Initial Release
- Real-time face recognition using webcam
- Support for Science, Arts, and Commerce student categories
- Automatic slideshow trigger based on majority detection
- Color-coded face bounding boxes (Green=Science, Red=Arts, Blue=Commerce)
- Live student count display per category
- Keyboard controls (q=quit, s=stop/reset, p=pause)
- Stability detection (10/15 frames threshold)
- 3-second poster rotation in slideshow
- Detailed README documentation
